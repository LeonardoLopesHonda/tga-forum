# TGA-Forum

This is a simple forum project that is made with Python/fastAPI for backend and Typescript/NextJS.

## Project Structure

```
tga-forum/
├── backend/
│   ├── .venv              # Python's virtual env
│   ├── main.py            # Backend routes and logic
│   └── requirements.txt   # Python's dependencies
├── frontend/
│   ├── src/               # Routes and pages folder
│   ├── tests/             # Tests folder
│   └── package.json       # Javascript's dependencies
└── README.md              # This file
```
